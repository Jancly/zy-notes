Arch LInux安装dde（Deepin Desktop Environment 深度桌面环境）

我一直比较推荐一些Linux新手使用Deepin Linux，因为我认为这种尽量的follow Windows的系统至少对于新手来说是比较的友好的，而且预装了QQ 火狐浏览器中文版，甚至还移植了像360安全浏览器这些，自带的应用商店等等，去掉了很多至少对于一般的Linux用户，初学者这些用不到的组件 还是相对很友好的，也就是开箱即用的。

当然我一直用deepin还有一个很重要的原因就是我认为这个系统的UI是非常好看的，可能是我接受不了LXDE Gnome这些的原生界面吧！至少我觉得如果说是还需要美化等等的这些真的是降低自己的效率的。

那么就开始咱们今天的内容吧！

因为dde和Gnome并不兼容，如果你原先安装的环境是Gnome的话，那么需要先移除所有gnome相关的包，另外，在这个过程中networkmanager也会被移除，一定要再次安装，否则会相当麻烦。

1.首先我们安装基本环境：

?
1
# sudo pacman -S deepin deepin-extra lightdm lightdm-deepin-greeter

 2.我们安装相关的依赖组件：

虽然说deepin-extra已经包含了deepin特有的一些应用，而且还挺简洁的，就和我上面说得，如果过多的组件会导致新手无从下手，不过，有些应用依旧缺失，需要补救一下。

?
1
2
# sudo pacman -S file-roller evince gedit thunderbird gpicview
# sudo pacman -S unrar unzip p7zip

 在安装完dde后，可能会遇到无法将语言改为中文的情况，只要删除～/.pam_environment文件即可。

?
1
# sudo rm ～/.pam_environment

 3.安装浏览器

浏览器我觉得都差不多，这个可以参考一下我上一个教程

Arch Linux安装Firefox 火狐中文版

4.最后

接下来我们就是直接重启我们的机器就可以了，在开机的时候，登陆界面会在左下角或者右下角会有一个切换桌面环境的选项，到时候可以注意一下。

登陆的时候使用非root用户登陆。
